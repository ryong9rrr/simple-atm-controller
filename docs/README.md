# ATM Controller Requirements

## ATM Controller

- [ ] ATM has **BankService**.

- [ ] ATM has **CashLocker**.

- [ ] insert **BankAccount**(BankCard, BankBook... etc)

- [ ] check correct PIN number with **BankService**.

- [ ] select Account.

- [ ] ATM work
  - [ ] confirm balance.
  - [ ] deposit money with **BankService, CashLocker**.
  - [ ] withdraw money with **BankService, CashLocker**.
  - [ ] end working
  - [ ] rework
